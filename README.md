## Installation
To get started with this project, you can either clone the repository or download the zip file. Once you have the code, navigate to the project directory in your terminal and run the following command to install the required dependencies:
```npm install```
## Usage
To start the development server, run the following command in your terminal:
```npm start```
This will start the development server at http://localhost:3000.
To build the project for deployment, run the following command:
```npm run build```
This will create a build folder with the optimized production build of the project.
